Description du Projet "dossierGit":
Ce projet contient 4 fichiers : 1 readme.txt et 3 fichiers contenant chacun du code.
Les fichiers sont :
---readme.txt
--index.html
--forum.php
--page_form.js
2/ Les étapes de la construction:
 a- Création du répertoire "dossierGit" avec la commande --mkdir--.
 b- Commande "git init" sur le répertoire "dossierGit" pour qu'ils soient pris en compte par git.
 c- Création des 4 fichiers (ceux cité plus haut) avec "touch".
 d-Modification avec vim de tous les fichiers puis "git add".
 e-Commit avec "git commit -m" et parfois avec "git -am".
 f- Commande git push -u origin https://github.com/TechLyon/dossierGit.git pour pushé mes fichiers au serveur github.
