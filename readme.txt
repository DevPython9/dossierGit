Description du répertoire "dossier git":
1/J'ai créé 3 fichiers contenant respectivement du code html, php et jquery.
Les fichiers sont :
--index.html
--forum.php
--page_form.js
2/Procédure:
J'ai créé le répertoire "dossier_git" avec la commande --mkdir--.
J'ai appliqué la commande git init sur celui-ci pour qu'ils soient pris en compte par git.
Puis j'ai créé 3 fichiers (ceux cité plus haut) avec "touch".
Après les avoir ouvert avec vim et mis du code dans chaun d'eux , j'ai appliqué la commande git add . sur tous les fichiers. Ensuite j'ai tout commité avec comme mesage "Code initial".
J'ai ensuite fait des commit comme le voulait le prof marc.
Ensuite j'ai utilisé la commande git push -u origin https://github.com/TechLyon/dossierGit.git pour pushé mes fichiers au serveur github.
